Below are three layout candidates for the controls (the 340px panel is tight), then the key states. Pick a layout and I'll build it.

Layout candidates

Same panel, same data — three ways to fit the controls. All shown in their default (no-filter) state so the list is identical; only the control treatment differs.

Reuses the existing pill idiom (the All courts / Sessions toggle). Tappable chips wrap across lines. Everything visible at a glance — no hidden state, no new control type. Costs the most vertical space (~3 rows).
